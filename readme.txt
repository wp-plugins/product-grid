	Awesome Product grid for WooCommerce.

== Description ==

Query Product and display on grid.

Use any where by short-codes and easy customize via short-code generator page, you can display or hide grid meta items like Product title, Product price, Product ratings and etc.


### Product Grid by http://paratheme.com





<strong>Plugin Features</strong><br />

* Fully responsive and mobile ready.
* Unlimited grid anywhere.
* Different Theme.
* Pagination support.
* Custom number grid Product per page.
* Display or hide Product meta's.
* Awesome hover effect.
* Social share buttons.
* Custom grid width.
* Custom grid thumbnail height.


== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Product Grid</strong>" activate it.<br />

After activate plugin you will see "Product Grid" menu at left side on WordPress dashboard click "New Product Grid" and use the options field "Post Grid"<br />

<br />
<strong>How to use on page or post</strong><br />
When Post Grid options setup done please publish Post Grid as like post or page<br />

and then copy shortcode from top of <strong>Post Grid Options</strong> `[product_grid  id="1234" ]`<br />

then paste this shortcode anywhere in your page to display grid<br />
